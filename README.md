# 🕵️ SQL Detective Academy

An interactive game that teaches SQL fundamentals through detective-themed challenges with competitive features!

## 🎮 Live Demo

[Play SQL Detective Academy](https://sql-detective-academy.streamlit.app)

## 📋 Features

### Core Platform
- ✅ Real SQLite database with crime investigation data
- ✅ Interactive SQL input with real query execution
- ✅ 5 progressive levels teaching SELECT, WHERE, ORDER BY/LIMIT, GROUP BY, JOIN
- ✅ Detailed feedback system with helpful hints
- ✅ Visual progress tracking with badges

### Competitive Features
- 🏆 **Global Leaderboard** - Compete with players worldwide
- ⏱️ **Time Trial Mode** - Race against the clock for bonus points
- 👥 **Multiplayer Races** - Compete against friends in real-time
- 🔥 **Streak System** - Maintain consecutive completions for bonuses
- 📊 **Performance Analytics** - Track your best times and stats
